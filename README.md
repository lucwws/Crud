## Installation💻

### Client

  1. `$ cd client`
  2. `$ npm install`
  3. `$ npm start`
  
### Server

  1. Check you DB in `server/index.js`
  2. `$ cd server`
  3. `$ npm install`
  4. `$ npm rum dev`
